# 控制面

控制面是 M01-M06 和 M09 后端部分的初始部署单元。

## 主要职责

- 可信身份上下文和 API 入口。
- 服务端策略授权和审计。
- Skill 契约、注册中心和候选路由。
- 持久化工作流、审批、幂等和补偿。
- DAG 校验、调度和制品引用。
- 向操作台提供强类型语义事件。

## 禁止事项

- 直接执行脚本。
- 直接使用目标系统长期凭据。
- 将授权决策交给模型。
- 跨模块直接访问数据库。

## 构建结构

控制面现在已经落为标准 Maven 聚合模块：

```text
control-plane/
|-- pom.xml
|-- bootstrap/
`-- modules/
```

其中 `bootstrap` 是未来控制面部署入口，`modules/` 下每个业务模块都是独立 Maven 子模块。

构建结构决策见 `docs/adr/0002-control-plane-build-structure.md`。
